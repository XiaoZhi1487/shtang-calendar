# 产品需求文档 (PRD)

## 1. 产品概述

一款具有独特间隔高亮功能的日历应用，采用"1001001"二进制高亮模式（每3天中第1天高亮），让用户能够直观地追踪周期性事件。应用包含日历、记账本、表格和日记四个核心功能模块。

## 2. 核心功能

### 2.1 功能模块

1. **日历页面（首页）**
   - 显示当前月份的日历视图
   - 按照"1001001"模式高亮特定日期（第1、4、7、10...天高亮）
   - 月份切换功能（上/下月）
   - 日历居中显示于屏幕中央

2. **记账本页面**
   - 收入/支出记录
   - 金额输入与分类
   - 历史记录列表

3. **表格功能页面**
   - 通用数据表格
   - 支持添加/编辑/删除行
   - 简单的数据管理

4. **日记功能页面**
   - 日期选择
   - 文本输入区域
   - 日记内容保存与展示

### 2.2 页面结构

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 日历 | 主视图 | 1001001间隔高亮日历，默认显示当月，居中布局 |
| 记账本 | 表单+列表 | 记录收支，支持分类 |
| 表格 | 数据表格 | 通用表格管理 |
| 日记 | 编辑器 | 日期+文本日记 |

## 3. 交互流程

### 3.1 页面导航
```
底部导航栏
├── 日历（默认首页）
├── 记账本
├── 表格
└── 日记
```

### 3.2 日历高亮逻辑
- 遵循"1001001"二进制模式
- 从每月1日开始计算，第1、4、7、10、13...天为高亮日
- 高亮样式：背景色高亮 + 圆角 + 轻微放大

## 4. 用户界面设计

### 4.1 设计风格
- **整体风格**: 简约现代，带有柔和的渐变色彩
- **主色调**: 深蓝灰 (#1e293b) 背景，带有温暖的琥珀色 (#f59e0b) 高亮点缀
- **按钮样式**: 圆角胶囊按钮，hover时有微妙的阴影提升
- **字体**: 使用系统无衬线字体栈，确保中文显示清晰
- **布局**: 底部标签导航，卡片式内容区域

### 4.2 页面设计

#### 日历页面
- 月份标题居中，大号字体
- 星期行：日 一 二 三 四 五 六
- 日期网格：7列布局
- 高亮日期：琥珀色背景，深色文字
- 非高亮日期：灰色文字，透明背景
- 今日：额外的边框强调

#### 导航栏
- 固定底部
- 4个图标按钮：日历、记账本、表格、日记
- 当前页面高亮显示

### 4.3 响应式设计
- 桌面端：日历和内容居中，最大宽度限制
- 移动端：全宽布局，触摸友好的按钮尺寸
